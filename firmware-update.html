GIGAIPC H310AI BIOS F5 Firmware UPDATE

Open Firmware Folder View Instruction (PDF)

Instructions Summary

  • Format USB to FAT32.
  • Copy BIOS file and flash tool to USB root directory.
  • Enter BIOS using [DEL] key on boot.
  • Go to Save & Exit > Launch EFI Shell.
  • Run fs0:\\flash.bat
  • Do not power off. System will reboot automatically.
  • Please ensure stable power during the update process.
