GIGAIPC H310AI BIOS F5 Firmware UPDATE

Open Firmware Folder
View Instruction (PDF)

Instructions Summary

  1. Format USB to FAT32.
  2. Copy BIOS file and flash tool to USB root directory.
  3. Enter BIOS using [DEL] key on boot.
  4. Go to Save & Exit > Launch EFI Shell.
  5. Run fs0:\flash.bat
  6. Do not power off. System will reboot automatically.

Please ensure stable power during the update process.
